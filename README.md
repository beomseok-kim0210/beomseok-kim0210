## 🧠 어떻게 일하는가

> “AI를 쓰는 개발자”가 아니라
> **“AI와 함께 사고하며 더 나은 결정을 내리는 개발자”**를 지향합니다.

- 문제 구조를 먼저 정의한 뒤 기술을 선택합니다.
- 기술적으로 가능하더라도 사용자 경험 기준에 맞지 않으면 과감히 제외합니다.
- 실험 → 검증 → 적용 판단을 반복합니다.
- 생성형 AI를 기능이 아니라 개발 전 과정의 파트너로 활용합니다.
